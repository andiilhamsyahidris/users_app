## What is Users App? 

### This application offers various features such as: 
- Show list of user, 
- Create user 
- Search user by name
- Sort user by alphabetically 

### Users App is built using flutter by applying : 
- Clean Architecture
- SOLID Principle 
- BloC
- GetX
- Get_it
